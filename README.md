# Chat en tiempo real Socket.Io 💻 🐶

Chat en tiempo real, creación de salas

## Installation

En el Json principal

```bash
npm install / npm i
```

## Uso -- Backend

```python
# ~/Chat-SocketIO (main)
cd Backend

# ~/Chat-SocketIO/Backend (main)
npm run dev
```
## Uso -- Frontend

```python
# ~/Chat-SocketIO (main)
cd rooms-chat

# ~/Chat-SocketIO/rooms-chat (main)
npm start 
```
